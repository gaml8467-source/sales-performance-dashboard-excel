# 📊 Excel Sales Dashboard

An interactive Excel Sales Dashboard built using **SQL Server**, **Power Query**, **Pivot Tables**, **Pivot Charts**, **Power Pivot**, and **Slicers**.

The project demonstrates the complete Excel Business Intelligence workflow, from extracting data from a SQL Server database to transforming, modeling, and visualizing sales data through interactive dashboards.

---

# 📌 Project Overview

This project consists of **two interactive dashboards** designed to analyze sales performance, customer behavior, products, territories, and order statistics.

The data was extracted from a SQL Server database, cleaned and transformed using Power Query, modeled inside Excel using Power Pivot, and presented through Pivot Tables, Pivot Charts, KPI Cards, and Slicers.

# 🛠 Data Preparation (ETL)

The ETL process included:

* Connecting Excel to SQL Server
* Importing sales-related tables
* Merging tables using Power Query
* Removing unnecessary columns
* Renaming fields
* Changing data types
* Handling missing values
* Creating a proper Date table
* Loading the cleaned model into Excel Data Model
